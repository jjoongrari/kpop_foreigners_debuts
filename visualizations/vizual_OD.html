import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

plt.style.use('seaborn-v0_8-whitegrid')

# цвета
purple = "#ddcaeb"
pink = "#FF4FA3"
blue = "#3A86FF"

# загрузка данных
df = pd.read_csv("kpop_group_debuts_2012_2025.csv")

# обработка
df["debut_date"] = pd.to_datetime(df["debut_date"], dayfirst=True)
df["year"] = df["debut_date"].dt.year
df["spotify_streams"] = pd.to_numeric(df["spotify_streams"], errors="coerce")

# разбиение по типу групп
df_foreign = df[df["has_foreigner"] == 1]
df_korean = df[df["has_foreigner"] == 0]

# динамика дебютов
debuts_total = df.groupby("year").size()
debuts_foreign = df_foreign.groupby("year").size()
debuts_korean = df_korean.groupby("year").size()

# медиана стримов
median_streams = df.groupby("has_foreigner")["spotify_streams"].median()

# страны
df_exploded = (
    df.dropna(subset=['foreign_countries'])
    .assign(country=df["foreign_countries"].str.split(", "))
    .explode("country")
)

df_exploded = df_exploded[df_exploded["country"] != "Unknown"]
top_countries = df_exploded["country"].value_counts().head(10)

Динамика дебютов K-pop групп (2012–2025)¶

График показывает изменение количества дебютов K-pop групп по годам с разделением на:

  • все группы
  • группы с иностранными участниками
  • полностью корейские группы

Визуализация позволяет проследить рост индустрии и оценить вклад интернациональных составов в общую динамику.

ax.yaxis.get_major_locator().set_params(integer=True)

Медианные значения Spotify streams¶

Для сравнения популярности групп используется медиана, а не среднее значение, поскольку распределение стримов содержит значительные выбросы.

График показывает различие между:

  • группами только с корейскими участниками
  • группами с иностранными участниками
